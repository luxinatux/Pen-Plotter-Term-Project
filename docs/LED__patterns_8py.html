Detailed Description

Changes LED patterns based on button input.

This script programs a nucleo to cycle between three different wave patterns(square, sin, and sawtooth) projected on a LED based on a button input. This is a video showing how the LED Light works https://cpslo-my.sharepoint.com/:v:/g/personal/dyruiz_calpoly_edu/EelOXbklHTRPhYRStjdc8MwBYotzM8h6hAvvF1_N4mXBbA?e=LdF9l4.
