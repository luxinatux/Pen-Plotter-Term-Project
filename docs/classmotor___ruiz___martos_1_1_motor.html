Detailed Description

A motor class for one channel of the DRV8847.

Objects of this class can be used to apply PWM to a given DC motor

Initializes and returns a motor object.

Sets up pin and timer objects to a motor. Initialization of pin object must be done in Main. Initialization of timer occurs in init.

Parameters
en_pinThe pyb.pin() object for the enable motor pin
in1pinThe pyb.pin() object for the PWM of the motor pin for forward direction.
in2pinThe pyb.pin() object for the PWM of the motor pin for reverse direction.
timerThe integer value of the timer channel wanted for the PWM.
